## 🛠️ Como rodar localmente

### Pré-requisitos

- Python 3.8 ou superior
- pip

```bash
git clone https://github.com/folopes/portfolio_st.git
cd portfolio_st
pip install -r requirements.txt
streamlit run app.py
```
